This project measures distance using the HC-SR04 ultrasonic sensor and displays it on a 16x2 LCD.

## Hardware
- Arduino Uno
- HC-SR04 ultrasonic sensor
- 16x2 LCD display
- Jumper wires
- Breadboard

## Wiring
- Trig → Arduino pin 13
- Echo → Arduino pin 12
- LCD pins → Arduino pins 11,10,9,2,3,4,5
